
# Asset Insights

這是一個現代化、專業的個人資產追蹤系統，支援股票、加密貨幣及銀行資產管理，並搭載 AI 財務分析建議。

## 功能特色
- **全球市場同步**：即時更新美股、台股及加密貨幣行情。
- **AI 財務顧問**：基於 Google Gemini 的深度投資組合分析。
- **多幣別支援**：支援 TWD, USD, CNY 隨時切換。
- **GitHub Pages 部署**：支援透過 GitHub Actions 自動化靜態部署。

## 開發與執行
1. 安裝依賴：`npm install`
2. 啟動開發伺服器：`npm run dev`
3. 靜態導出：`npm run build` (產出至 `out` 資料夾)
